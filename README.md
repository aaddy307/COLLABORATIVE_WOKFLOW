# DESCRIPTION OF THE TASK

In this task, I simulated a collaborative workflow in Figma by creating and managing a shared design project. The objective was to understand how teams work together in real-time using design collaboration tools.

I started by creating a multi-page project in Figma and organizing different design tasks such as logo design, e-commerce UI, and a voice-controlled app interface. After structuring the project, I shared it with team members to enable collaboration.

The workflow included the following key activities:

- **Project Sharing:** Shared the Figma file with team members to allow real-time collaboration  
- **Commenting System:** Added comments on designs to suggest improvements and communicate feedback  
- **Design Feedback:** Reviewed UI elements and suggested enhancements such as improving button contrast, refining error messages, and enhancing icon visibility  
- **Version Control:** Managed multiple design versions including initial layout, UI improvements, and final design  
- **Iteration Process:** Continuously refined the design based on feedback and updates  

During this task, I focused on:

- Effective team communication through comments  
- Maintaining design consistency across updates  
- Organizing files and layers for better collaboration  
- Tracking changes using version history  

This task helped me understand real-world collaborative design workflows, including teamwork, feedback cycles, and version management using Figma.
